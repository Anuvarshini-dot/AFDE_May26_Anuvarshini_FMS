This folder contains UI screenshots for Phase 2 of the Feedback Management System, which introduced role-based access, an ETL pipeline for bulk data import, and a program-wise analytics dashboard.

---

## Phase 2 Features Covered

- **Role-based access** — Admin and User roles with separate navigation and permissions
- **ETL pipeline** — bulk import of feedback records from CSV or Excel files with validation, deduplication, and run history
- **Program filter** — dashboard and analysis page both support filtering by program via a dropdown
- **Analysis page** — program-wise rating distribution, ranking table, and per-program feedback insights derived from user comments
- **Downloadable reports** — CSV export filtered by program or across all programs
